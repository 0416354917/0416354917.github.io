Stock Maximize

July 1, 2016

Source: HackerRank

Analysis:

Dynamic Programing. Time complexity is O(n).

Solution: 

              
t = int(raw_input())
for t_i in xrange(t):
    n = int(raw_input())
    price = map(int, raw_input().split())
    
    profit = 0
    index  = n - 1
    peek = price[index]
    index -= 1
    while index >= 0:
        while index >= 0 and price[index] >= peek:
            peek = price[index]
            index -= 1
        if index == -1:
            break
        else:
            local_profit = 0
            while index >= 0 and price[index] < peek:
                local_profit += (peek - price[index])
                index -= 1
            profit += local_profit
    print profit
